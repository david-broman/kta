Introduction
------------

The 'precision timed compiler (ptc)' is a tool chain designed for
compiling and analyzing programs with respect to real-time. The ptc
compiler uses 'clang' as front-end for compiling a C/C++ program into
LLVM assembly code.

Timing Analysis Input Format
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If a filename with ending +.ta+ (for 'timing analysis') is supplied as
arguments to the ptc compiler, the compiler will perform timing
analysis as part of the compilation processes.
